# Divide-and-conquer based Large-Scale Spectral Clustering

## Datasets
Five real world datasets and five synthetic datasets:

- USPS 
- PenDigits 
- Letters 
- MNIST 
- Covertype 
- Three Spirals-60K (TS-60K)
- Two Moons-1M (TM-1M)
- Three Circles-6M (TC-6M)
- Circles and Gaussian-10M (CG-10M) 
- Flower-20M (FL-20M)
